# 💼 Portfolio Website

This is a simple and responsive personal portfolio website to showcase your skills, projects, and contact information. It is built using basic web technologies and is easy to customize.

## 🛠️ Technologies Used

- **HTML** – Provides the structure of the website.
- **CSS** – Adds styling and layout to make the site visually appealing.
- **JavaScript** – Adds interactivity such as navigation effects and dynamic elements.
